API Reference

Overview

[Class] Component

Component is the base class for all controls (including Canvas). All other controls will inherit its properties, methods and events.

If you initiate a Component object and add it into the canvas, nothing will be renderred as Component doesn't has "render()" method itself.

You may use Panel instead or create your own class inheriting from Component, and implement the "render()" method.

Please reference to the "Toolbar" example of ResponsiveApp.

Properties
isEnabled Default Value: true

When setting this property, the value will be propagated to all the controls in the children tree.

isVisible Default Value: true

When setting this property, the value will be propagated to all the controls in the children tree.

fillStyle Default Value: null
strokeStyle Default Value: '#000'
lineWidth Default Value: 0

The border will be drawn only if "lineWidth" not equals 0 and "strokeStyle" is not null.

Methods
getProp(key) Get the value of a property by key.
setProp(key, vakue) Set the value of property by key.

[Class] Canvas

Canvas is the container for all controls.
It's derived from Component.

Constructor
var canvas = new Canvas(width, height, isFullScreen);

width: Width of the canvas.

height: Height of the canvas.

isFullScreen: If the canvas is responsive. (Default value: false)
Events
onWillPaint(context) Before the canvas painting.
onDidPaint(context) When the canvas is painted.
Example 
var canvas = new Canvas(500, 100);
canvas.setProp('lineWidth', 1);
canvas.setProp('strokeStyle', '#333');
canvas.setProp('fillStyle', 'RGBA(0, 0, 0, 0.5)');
var wrapper = document.getElementById('brief-look-wrapper');
canvas.renderInto(wrapper);

[Class] Panel

Panel renders a rectangle on the canvas. It can have border or background color.
It's derived from Component.

Constructor
var panel = new Panel(left, top, width, height, id);

left: Left of the control, which is relative to its parent.

top: Top of the control, which is relative to its parent.

width: Width of the control.

height: Height of the control.

id: Optional, and you may leave it empty.
Example 
var panel = new Panel(10, 10, 500, 100);
panel.setProp('lineWidth', 1);
panel.setProp('strokeStyle', '#333');
panel.setProp('fillStyle', 'RGBA(0, 0, 0, 0.5)');
canvas.controls.add(panel);

[Class] Label

Label renders text on the canvas. It can have border or background color.
It's derived from Panel.

Constructor
var label = new Label(left, top, width, height, text, id);

left: Left of the control, which is relative to its parent.

top: Top of the control, which is relative to its parent.

width: Width of the control.

height: Height of the control.

(If "isSizeFixed" equals false, width and height will be calculated according to the text content, and values passed in will be ignored.)

text: The text content to be renderred.

id: Optional, and you may leave it empty.
Properties
fontSize Default Value: 12
fontFace Default Value: 'Arial, Helvetica, sans-serif'
fontColor Default Value: '#000000'
text

The text content to be renderred.

horizontalAlign Default Value: 'left'

Value Options: 'left' | 'center' | 'right'

verticalAlign Default Value: 'top'

Value Options: 'top' | 'middle' | 'bottom'

isSizeFixed Default Value: false

If "isSizeFixed" equals false, width and height will be calculated according to the text content, and values passed in will be ignored.

Example 
var label = new Label(10, 10, 0, 0, 'Label Text', 'label1')
label.setProp('isVisible', false);
label.setProp('fontSize', 20);
canvas.controls.add(label);

[Class] Button

Button is derived from Label.

Constructor
var button = new Button(left, top, width, height, text, id);

left: Left of the control, which is relative to its parent.

top: Top of the control, which is relative to its parent.

width: Width of the control.

height: Height of the control.

text: The text content to be renderred.

id: Optional, and you may leave it empty.
Properties
isOn Default Value: false
radius Default Value: 0
focusStyle Default Value: 
{
  fillStyle: '#337AB7',
  strokeStyle: '#337AB7',
  lineWidth: 1,
  fontColor: '#fff'
}
Events
onTap(left, top) The event for button click with the position on which the button is clicked.
Example 
var button = new Button(10, 10, 130, 32, 'Save');
button.setProp('focusStyle', {
  'fontColor': '#f00',
  'strokeStyle': '#f00'
});
button.onTap.add(function(left, top) {
  console.log('Button has been clicked at [' + left + ', ' + top + ']')
});
canvas.controls.add(button);
